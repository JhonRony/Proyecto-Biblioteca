# Sistema de Gestión de Biblioteca

Un sistema completo para la gestión de una biblioteca que permite administrar libros, alumnos, categorías y préstamos.[web:21]

## Características Principales

- Gestión de Libros (agregar, editar, eliminar)
- Gestión de Alumnos
- Categorización de Libros
- Sistema de Préstamos y Devoluciones
- Panel de Estadísticas
- Interfaz intuitiva y fácil de usar[web:21]

## Requisitos

- Servidor web (XAMPP, WAMP, etc.)
- PHP 7.4 o superior
- MySQL 5.7 o superior
- Navegador web moderno[web:21]

## Instalación

1. Clona el repositorio:
git clone https://github.com/JhonRony/Proyecto-Biblioteca.git

2. Importa la base de datos:
- Abre phpMyAdmin
- Crea una nueva base de datos llamada `biblioteca`
- Importa el archivo `database/biblioteca.sql`

3. Configura la conexión:
- Edita el archivo `config/database.php` con tus credenciales

4. Accede al sistema:
- Abre tu navegador
- Navega a `http://localhost/PROYECTO-BIBLIOTECA`[web:21][web:31]
